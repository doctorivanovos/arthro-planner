# Планирование операционного дня — артроскопия

Приложение для планирования операционного дня артроскопического отделения
(Клиника высоких медицинских технологий им. Н. И. Пирогова СПбГУ).

**Открыть приложение:** https://USERNAME.github.io/arthro-planner/

## Что умеет

- Считает, сколько операций влезает в день с учётом реальных ограничений:
  столы, артроскопические стойки, оптики, шейверы, наборы инструментов,
  хирурги и их навыки, анестезиологическая бригада, стерилизация 90 минут.
- Диаграмма дня по столам с перетаскиванием операций (мышью и пальцем).
- План на неделю: у каждой даты — свой день.
- Поимённый список пациентов или планирование «по числам»; свои
  (нешаблонные) операции текстом.
- Режим «остаток дня»: журнал выполненного + живые таймеры стерилизации.
- Списание имплантов на пациента и сводка расхода за день.
- Срезы для бригады: операционная м/с, ЦСО, анестезиолог, по столам —
  печать, копирование, отправка в мессенджер.
- Сценарии «что сломается», анализ узких мест, шаблоны дня.
- Экспорт/импорт всех данных (страховка от очистки браузера).

## Как пользоваться

Просто откройте ссылку — установки не нужно. Все данные хранятся локально
в вашем браузере и никуда не передаются. У каждого пользователя — свои
настройки отделения, шаблоны и планы.

Локальный запуск: `node server.js` → http://localhost:8777

## Технически

Один файл `index.html` без зависимостей и серверной части.
